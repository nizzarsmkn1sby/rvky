# Project Setup: Laravel + Breeze

This project has been set up with Laravel and Breeze.

## Configuration
- **Database**: `RIVE` (MySQL)
- **Auth Stack**: Breeze (Blade)

## Note on Assets
The assets were not built because the current Node.js version is incompatible with the latest Vite. 
Please upgrade Node.js to version 20 or higher and run:
```bash
npm install
npm run build
```
